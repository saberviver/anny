Meus trabalhos

Logo pra uma marca local

Desenhei um logo pra uma marca de roupas sustentáveis. Usei ferramentas de design gráfico como Illustrator e Photoshop pra criar algo simples e conectado à natureza. Foi um projeto rápido, mas o cliente achou que ficou adequado pra identidade visual que queriam.

Site de portfólio pra uma fotógrafa

Desenvolvi um site de portfólio usando HTML, CSS e JavaScript. Fiz um layout bem clean pra destacar as fotos e usei JavaScript pra criar uma galeria com transições suaves e efeitos interativos. Também deixei o site responsivo, funcionando bem no celular e no desktop. O resultado ficou funcional e visualmente agradável.

Sistemas de Reservas pra um salão de beleza

Criei um sistema de reservas online usando HTML e CSS pra interface e PHP no backend pra processar os dados. Fiz uma página onde os clientes podiam escolher horários disponíveis e receber confirmação por e-mail. O dono do salão gostou porque simplificou o processo de agendamento.

Loja no Shopify pra vender bijuterias artesanais

Configurei uma loja Shopify e personalizei o tema com HTML e CSS pra deixá-la mais atrativa. Adicionei detalhes como banners animados e estilização personalizada nos botões. Também configurei as páginas de produto e checkout pra garantir uma navegação fácil. Depois disso, a loja começou a funcionar bem e atraiu os primeiros clientes.

Jogo 3D no Unity

Desenvolvi um jogo 3D no Unity, focado num personagem que desvia de obstáculos num cenário futurista. Trabalhei com C# pra programar as mecânicas e usei ferramentas do próprio Unity pra criar os modelos e animações. Foi um projeto experimental, mas deu pra aprender bastante sobre física e interação em jogos.
